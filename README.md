# flux-apps

Real FLUX VM bytecode programs demonstrating agent capabilities.

## Programs

| Program | Description | Key Opcodes |
|---------|-------------|-------------|
| [Decision Tree](docs/decision-tree-design.md) | 4-action weighted choice | RAND_WEIGHTED, CONF_GET, HABITUATE |
| [Prime Sieve](docs/prime-sieve-design.md) | Sieve of Eratosthenes | MEMFILL, MEMRANGE_MIN, LOOP |

## The Deeper Connection

A decision tree weighing instinct against energy against confidence is a mind making choices, not a calculator computing values. These programs prove the FLUX ISA can express genuine agency.
